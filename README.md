# 📡 TelNet - Conexión Segura

Prototipo de aplicación móvil para la gestión de servicios de telecomunicaciones (internet, telefonía y cable). Desarrollado como proyecto universitario para la Universidad Tecnológica del Perú (UTP).

---

## Tecnologías utilizadas

- **React 19** — Librería de interfaces de usuario
- **Vite 8** — Herramienta de desarrollo ultrarrápida
- **React Router DOM v6** — Navegación entre pantallas
- **Recharts** — Gráficas de velocidad en tiempo real
- **CSS en línea** — Estilos personalizados sin dependencias externas

---

## Pantallas de la aplicación

| # | Pantalla | Descripción |
|---|----------|-------------|
| 1 | **Login** | Inicio de sesión con credenciales |
| 2 | **Dashboard** | Resumen del estado del servicio |
| 3 | **Monitoreo** | Gráfica de velocidad en tiempo real |
| 4 | **Servicios** | Gestión de servicios contratados |
| 5 | **Incidencias** | Sistema de tickets y reportes |
| 6 | **Soporte** | Chat de soporte técnico virtual |
| 7 | **Dispositivos** | Dispositivos conectados a la red |
| 8 | **Cuenta** | Perfil, facturación y configuración |

## Instalación y ejecución

### Requisitos previos
- Node.js v18 o superior
- npm v9 o superior

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/NicolaSKY/APP_ConexionSegura.git

# 2. Entrar a la carpeta
cd APP_ConexionSegura/telnet-app

# 3. Instalar dependencias
npm install

# 4. Ejecutar en modo desarrollo
npm run dev
```

Abrir en el navegador: `http://localhost:5173`
